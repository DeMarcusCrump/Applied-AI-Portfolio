# ITAI-1378: Computer Vision & Artificial Intelligence

## Course Overview

This course covers the fundamental and advanced concepts of computer vision, including image processing, feature detection, object recognition, and deep learning for visual pattern recognition. Projects demonstrate proficiency in building computer vision systems using modern deep learning frameworks.

## Key Learning Outcomes

- Image Processing & Enhancement
- Feature Detection & Extraction
- Convolutional Neural Networks (CNNs)
- Image Classification
- Object Detection & Recognition
- Transfer Learning Techniques
- Model Training & Optimization
