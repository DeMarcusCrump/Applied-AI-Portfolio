# ITAI-2372: Artificial Intelligence Applications

## Course Overview

This course focuses on building production-ready AI applications with real-world impact. Projects demonstrate the integration of advanced AI technologies including Large Language Models (LLMs), blockchain technology, and intelligent systems design into functional web applications.

## Key Learning Outcomes

- AI API Integration (Groq, OpenAI-compatible endpoints)
- Multi-Agent AI Architecture Design
- Blockchain & Smart Contract Implementation
- Full-Stack Web Development with AI
- Production-Ready Application Deployment
- Real-World Problem Solving with AI

## Course Projects

### 1. [Aura Intelligence](./Aura-Intelligence/)
**AI-Powered Fragrance Marketing Platform**

An enterprise-grade application that transforms fragrance descriptions into compelling marketing content using advanced LLMs.

**Technologies:** Python, Flask, Groq API (Llama 3.3), JavaScript, Server-Sent Events  
**Key Features:**
- Dual-mode AI (Storyteller & Curator)
- Multi-agent consultation system
- Real-time streaming responses
- Luxury e-commerce optimization

---

### 2. [AgentHub](./AgentHub/)
**Blockchain-Powered AI Agent Marketplace**

A decentralized marketplace for autonomous AI agents with built-in trust infrastructure and automated quality verification.

**Technologies:** Python, Custom Blockchain, ML Validation, Flask  
**Key Features:**
- SHA-256 cryptographic blockchain
- Smart contract escrow system
- 92% accurate AI validation
- Immutable audit trail

---

### 3. [Asthma & Allergy Co-Pilot](./Asthma-Allergy-Co-Pilot/)
**AI-Driven Respiratory Health Companion**

An intelligent health management system providing personalized asthma and allergy assistance with real-time monitoring.

**Technologies:** Python, AI/ML, Health Data Analytics, Full-Stack Development  
**Key Features:**
- Real-time air quality monitoring
- Personalized symptom tracking
- AI-powered health recommendations
- Research-backed interventions

---
