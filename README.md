## Featured Courses & Projects

### Artificial Intelligence Applications (ITAI-2372)

**Course Focus:** Building production-ready AI applications with real-world impact

#### [Aura Intelligence](./ITAI-2372-AI-Applications/Aura-Intelligence/)
An AI-powered fragrance marketing platform leveraging Groq's advanced language models to transform technical fragrance notes into narrative-driven product descriptions.

**Key Features:**
- Dual-mode AI system: Storyteller Mode (creative content) & Curator Mode (conversational recommendations)
- Multi-agent architecture with 3 specialized AI agents
- Real-time streaming responses using Flask Server-Sent Events
- Production-ready web application with luxury design system

**Technologies:** Python, Flask, Groq API (Llama 3.3), JavaScript, HTML/CSS

---

#### [AgentHub](./ITAI-2372-AI-Applications/AgentHub/)
A blockchain-powered autonomous agent marketplace with built-in trust infrastructure for AI-driven work verification.

**Key Features:**
- Custom blockchain implementation with SHA-256 cryptographic hashing
- Smart contract-based escrow payment system
- AI validator for automated quality enforcement (92% accuracy)
- Immutable transaction history and transparent reputation system

**Technologies:** Python, Blockchain, Machine Learning, Transformer-based Validation, Flask

---

#### [Asthma & Allergy Co-Pilot](./ITAI-2372-AI-Applications/Asthma-Allergy-Co-Pilot/)
An AI-driven respiratory health companion providing personalized asthma and allergy management assistance.

**Key Features:**
- Real-time air quality monitoring and alerts
- Personalized symptom tracking and analysis
- AI-powered health recommendations
- Comprehensive research-backed health companion

**Technologies:** Python, AI/ML, Health Data Analytics, Full-Stack Development
